La programmation, c'est une matière très importante puisqu'elle continuera durant les 4 années. Nous voyons cette année les notions telles que les variables, les boucles, opérateurs logiques ou débugage. Nous travaillons grâce à structorizer pour schématiser l'algorithme et python pour programmer.

Toutes ces compétences doivent être maitrisées en structorizer et en python

  1. Étape 1

    Prérequis

    1. Résoudre des situations simples dans code.org
    2. Les 4 premieres leçons de l'express course ne doit poser aucun soucis.
    3. Compréhension de consignes et le plaisir de chercher.
    4. Télécharger et lancer structorizer sur leur machine
    5. Aller sur le site pour Python
  2. Étape 2

    L'algorithmie

    1. Pouvoir mettre des instructions l'une en dessous de l'autre face à un problème en français
    2. Présenter un structorizer pour lister la résolution d'un problème
  3. Étape 3

    Les variables

    1. Écrire une variable selon le langage
    2. Définir le type de variable en fonction de la situation
    3. Affecter une valeur à une variable
    4. Manipuler ses variables pour faire des calculs avec celles-ci
    5. Donner la valeur d'une variable
    6. Utiliser la concaténation pour construire des phrases mélangeant textes et variables
    7. Comprendre l'opérateur modulo
  4. Étape 4

    Les fonctions

    1. Utiliser les fonctions pour des opérations mathématiques
    2. Utiliser les fonctions sur des chaines de caractères
  5. Étape 5

    Les boucles

    1. Maitriser les exercices de code.org en lien avec les boucles.
    2. Maitriser la boucle for
    3. Maitriser la boucle while
    4. Pouvoir imbriquer différentes boucles utilisées ci-dessus
    5. Ajouter des opérateurs logiques dont "et", "ou" et "non"
  6. Étape 6

    En complément

    1. Commenter votre code pour qu'il puisse être utilisé par une autre personne
    2. Face à une situation complexe, pouvoir chercher et proposer des solutions
    3. Corriger une analyse ou séquence défaillante sans spécialement la lancer
    4. Maitriser la boucle until
